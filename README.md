# DaysOfCoding2023SignUp

## Setup instructions
Suggested editor: VScode with the `live server` extension

## Python environment

**Windows, cmd**
```powershell
> python -m venv venv
> .\venv\Scripts\activate.bat
> pip install -r requirements.txt
```

**Linux, bash**
```powershell
$ python -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```
